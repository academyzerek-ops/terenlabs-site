# Мастер-план внедрения (синтез 4 исследований)

> 2026-06-07. Источники: 01-award-winners, 02-saas-edtech, 03-dark-immersive, 04-typography-grids.
> Принципы: контент никогда не скрывается эффектом · всё проверяется в WebKit ·
> prefers-reduced-motion · без mix-blend-mode и живого blur>80px.

## Волна 1 — фундамент (риски и база)
- [x] ~~mix-blend-mode в cursor-glow~~ → заменить на rgba-слой без бленда (iOS 26 регрессия!)
- [ ] Шум-тайл 4% на тёмных секциях (base64, отдельный псевдоэлемент)
- [ ] Виньетка hero (inset radial, 35–55%)
- [ ] Текст: body #D6E4EC-класс ✓ (уже), вторичный ≥55% ✓, бирюза мелкая → #33C9D2
- [ ] Типографика: display ls -0.01em (не -0.02), body lh 1.6 контента ✓ (embed-web 1.85 ok)

## Волна 2 — главная по формуле Linear+Khan+Brilliant
- [ ] Нумерация секций «01/02/03» JetBrains Mono + тонкие линии (консультантский вид)
- [ ] Секция-манифест «Почему бесплатно» (Khan: бесплатность = миссия, иначе читается дешёвкой)
      + эмоциональный экран «Защита от потерь, не успешный успех» (Raycast)
- [ ] **Живой вопрос теста на лендинге** (Brilliant: продукт и есть демо) — 1 реальный вопрос
      из банка t1-a04, после ответа → CTA «Пройти все 42»
- [ ] CTA-аудит: первое лицо + исход («Посчитать мой бизнес»), 1 главный на экран

## Волна 3 — микровзаимодействия (Safari-надёжные)

- [ ] Hover-инверсия строк там, где списки
