# «Дорогая тьма»: тёмные/immersive сайты 2025–2026

> Исследование 2026-06-07: 14 галерей, 60+ сайтов (Linear, Vercel, Igloo Inc, Fey, Qase,
> Convex Seascape, The Deep Sea, Iutopi…). Ключевое — ниже; Safari-риски критичны.

## Слои тьмы
1. **Лестница 4–6 ступеней светлоты** вместо теней: фон L8–12% → карточки L14–16% → модалки L18–20%.
   Для океана: #06141F (дно) → #081D30 → #0D2B45 (карточки) → #14395C (hover).
2. **Цветная тьма дороже нейтральной** (Igloo Inc взял SOTY на двух сине-серых). Navy — преимущество.
3. Чистый #000 — только одно место («дно колодца» hero/футера).
4. **Скролл = погружение**: тон фона меняется по глубине страницы (у нас уже есть на /levels).

## Глубина без 3D
5. **Свет сверху**: radial-gradient бирюзы ≤15% у верхней кромки (физически мотивирован — солнце сквозь воду).
6. Aurora-пятна: 2–4 размытых пятна, медленный дрейф 20–30s. Blur >80px — Safari-яма: ЗАПЕКАТЬ в PNG.
7. **Шум 3–6%** поверх градиентов — убивает banding (признак дешёвой тьмы) + плёночная текстура.
   Тайл 128×128 base64 (TG WebView игнорит preload!).
8. **Виньетка**: inset-затемнение краёв 35–55%, едва заметная.
9. Параллакс low-poly силуэтов: дальний слой темнее+размытее (атмосферная перспектива).
10. Лучи света: наклонные linear-gradient ≤10% opacity.
11. **1px границы светлее фона** вместо теней; верхняя кромка ярче (свет сверху).
12. Inner glow: inset 0 1px 0 rgba(255,255,255,.06); цветное свечение — только hover/ключевые.
13. Градиентная рамка с бегущим бликом — только 1–2 hero-элемента.
14. Solid vs hollow текст: контурные слова-водяные знаки (opacity 4–6%).

## Пирамида интенсивности
- Едва заметное постоянное (90%): шум, виньетка, дрейф, границы — «зритель не видит приёма, видит качество»
- Среднее по скроллу: смена тона, параллакс, reveal
- Акцент: ОДИН на экран
15. НЕ делать: частицы по всему экрану, курсор-шлейфы, автозвук, >1 вау-эффекта на вьюпорт.
16. Частицы если очень надо: 6–10 шт CSS-only, ≤0.25 opacity, только hero.

## Читаемость
18. Заголовки #F5F7FA; body **#D6E4EC** (холодный off-white с каплей бирюзы), не чистый белый (halation).
19. Светлый текст худеет: body 400→450/500, ls +0.01em, lh 1.5–1.6, мера 60–70ch.
20. **Лонгрид-зона спокойнее hero**: без aurora, текст никогда поверх градиентных пятен.
21. Вторичный текст ≥ rgba(255,255,255,.55).
22. Бирюза #00B7C2 на #0D2B45 = 4.6:1 — только крупно; мелкое — осветлять до #33C9D2.

## Переходы тёмное↔светлое
23. Градиентный шлюз 120–240px + шум.
24. Или смена --bg по скроллу (стыков нет вообще).
25. Светлая секция как «лист-объект» на тьме (radius 24+, свечение по краю) — для PDF/прайса.
26. Инверсия хрома на стыке (хедер) — забытая инверсия = типовой ляп.

## Акцент
27. 60-30-10: плотная бирюза ≤5% площади экрана.
28. **Формула: плотная бирюза — мелко (CTA/линии/иконки), прозрачная — крупно (свечения).**
29. Куда НЕ класть: фоны секций, body-текст, рамки всех карточек подряд.

## Картинки на тьме
30. Фото: brightness(.85–.9) contrast(1.05); скриншоты продукта — тёмные.
31. Рамка-носитель: подложка+1px свет+r12–16. Голая картинка на фоне — нет.
32. Свечение-эхо за артом: radial доминантного цвета, blur запечь.
33. **Low-poly арты растворять маской в фон** (radial gradient mask) — «существо в воде»,
    не «картинка в посте». -webkit-mask-image обязателен.

## Океанские находки
35. Скролл-погружение с этапами (Ракушка у поверхности → Акула на глубине) ✓ уже сделано
36. Холод = холодный свет, не голубой фон. Один тёплый акцент («фонарь удильщика») — резерв.
37. Двух цветов достаточно: больше ступеней между navy и бирюзой, не третий цвет.

## SAFARI — КРАСНАЯ ЗОНА
1. **mix-blend-mode НЕ ИСПОЛЬЗОВАТЬ** — регрессия iOS 26 (заливка плоским цветом) + конфликт
   с -webkit-backdrop-filter. Всё то же — полупрозрачными rgba-слоями.
2. filter: blur(80–120px) живьём — тормозит; запекать пятна в PNG/WebP, анимировать только transform/opacity.
3. backdrop-filter — только узкая шапка, с -webkit-.
4. mask-image — только image/gradient-маски с -webkit- (SVG url(#id) не работает).
5. background-blend-mode игнорится при repeating — шум отдельным псевдоэлементом.
6. Noise-тайл инлайнить base64 (TG WebView).
7. Деградация честная: без эффектов сайт = чистый navy (additive-слои).

## Сводный рецепт
Лестница #06141F→#081D30→#0D2B45→#14395C, границы rgba(255,255,255,.08–.14).
Текст #D6E4EC/#F5F7FA/60%. Атмосфера: свет сверху ≤15% + запечённые пятна + шум 4% + виньетка.
Бирюза плотная ≤5%, прозрачная в свечениях. Лонгриды — тихая поверхность.
Без blend-modes, без живого blur>80, всё за prefers-reduced-motion.
